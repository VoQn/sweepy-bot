# Development workspaces for `@sweepy-bot`

## これなに？

[sweepy-bot](https://github.com/VoQn/sweepy-bot) の開発支援系のパッケージをまとめたワークスペースです

## packages

### **`@sweepy-bot/babel`**

**`.js` を [babel](https://babeljs.io/)** でよしなに解釈する用のやつ

### **`@sweepy-bot/eslint-config`**

**Linter : [eslint](https://eslint.org/)** 用のプリセット

### **`@sweepy-bot/jest`**

**Test Runner : [jest](https://jestjs.io/ja/)** の基本設定など

### **`@sweepy-bot/prettier`**

**File Formatter: [prettier](https://prettier.io/)** の共通設定やらプラグインやら

### **`@sweepy-bot/typescript`**

**Transpiler: [Typescript](https://www.typescriptlang.org/)** の共通コンパイル設定など
